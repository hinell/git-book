## Usage
Previously, ensure that you have installed
[iojs](https://iojs.org/en/index.html) and [bower](http://bower.io/).

## Deploy and Usage

```
$ bower install
$ npm   install
$ npm   start
```


Then open up [127.0.0.1](http://127.0.0.1) page in your browser.
